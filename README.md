# Projects

A collection of hands-on networking and cloud labs demonstrating practical knowledge of infrastructure, networking, and enterprise technologies.

## What's Included

* Network design and implementation labs
* Routing and switching configurations
* VLANs and inter-VLAN communication
* Dynamic routing protocols (e.g., OSPF)
* Cloud and infrastructure-related projects
* Topology diagrams and configuration files

Each project is self-contained and includes objectives, network diagrams, and configurations used to build and validate the solution.

This repository serves as a portfolio of practical labs and technical concepts developed to strengthen and showcase my skills in networking, cloud, and IT infrastructure.
